PowerShell Commands

A comprehensive list of PowerShell commands for system administration and Active Directory management.

Command Description Example Action
Get-NetAdapter Displays network adapters Get-NetAdapter
Get-ADUser Retrieves Active Directory user accounts Get-ADUser -Filter * -Properties *
New-ADUser Creates a new Active Directory user New-ADUser -Name "John Doe" -SamAccountName "jdoe" -UserPrincipalName "jdoe@domain.com" -Enabled $true
Get-ADGroup Retrieves Active Directory groups Get-ADGroup -Filter *
Add-ADGroupMember Adds members to an Active Directory group Add-ADGroupMember -Identity "GroupName" -Members "UserName"
New-ADOrganizationalUnit Creates a new Organizational Unit in AD New-ADOrganizationalUnit -Name "Department" -Path "DC=domain,DC=com"
Set-ADAccountPassword Sets the password for an AD account Set-ADAccountPassword -Identity "UserName" -Reset -NewPassword (ConvertTo-SecureString -AsPlainText "NewPassword" -Force)
Enable-ADAccount Enables an AD account Enable-ADAccount -Identity "UserName"
Disable-ADAccount Disables an AD account Disable-ADAccount -Identity "UserName"
Get-Service Gets the services on a computer Get-Service | Where-Object {$_.Status -eq "Running"}
Get-Process Lists all running processes Get-Process | Sort-Object CPU -Descending
Test-NetConnection Tests network connectivity Test-NetConnection -ComputerName google.com -Port 443
Get-DnsClientCache Displays DNS cache contents Get-DnsClientCache
Get-EventLog Retrieves event logs Get-EventLog -LogName System -Newest 10
Get-GPO Gets Group Policy Objects Get-GPO -All
Get-Disk Shows disk information Get-Disk | Select-Object Number, Size, PartitionStyle
Get-ComputerInfo Displays detailed system information Get-ComputerInfo | Select-Object WindowsVersion, OsName, OsArchitecture
Invoke-Command Runs commands on local/remote computers Invoke-Command -ComputerName "Server01" -ScriptBlock { Get-Service }
Get-ADDomain Gets domain information Get-ADDomain | Select-Object Forest, DomainMode, PDCEmulator
Get-ADComputer Lists domain computers Get-ADComputer -Filter * -Properties *
Get-NetFirewallRule Lists firewall rules Get-NetFirewallRule | Where-Object {$_.Enabled -eq 'True'}
New-NetFirewallRule Creates a new firewall rule New-NetFirewallRule -DisplayName "Allow Port 80" -Direction Inbound -Action Allow -Protocol TCP -LocalPort 80
Get-Acl Gets file/folder permissions Get-Acl -Path "C:\folder" | Format-List
Set-Acl Sets file/folder permissions $acl = Get-Acl "C:\folder"; Set-Acl -Path "C:\newfolder" -AclObject $acl
Get-ScheduledTask Lists scheduled tasks Get-ScheduledTask | Where-Object {$_.State -eq 'Ready'}
Register-ScheduledTask Creates a new scheduled task $action = New-ScheduledTaskAction -Execute 'Powershell.exe'; Register-ScheduledTask -Action $action -TaskName "MyTask"
Get-WindowsFeature Lists Windows features Get-WindowsFeature | Where-Object {$_.InstallState -eq 'Installed'}
Get-ItemProperty Gets registry values Get-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion"
Get-Counter Gets performance counter data Get-Counter '\Processor(_Total)\% Processor Time'
Backup-GPO Backs up Group Policy Objects Backup-GPO -All -Path "C:\GPOBackups"
Resolve-DnsName Performs DNS name resolution Resolve-DnsName -Name "google.com" -Type A
Get-NetTCPConnection Shows TCP connections Get-NetTCPConnection | Where-Object State -eq 'Established'
Get-ChildItem Cert: Lists certificates Get-ChildItem Cert:\LocalMachine\My
Get-WindowsUpdate Lists available Windows updates Get-WindowsUpdate -MicrosoftUpdate
Enter-PSSession Starts interactive session with remote computer Enter-PSSession -ComputerName "Server01"
Write-EventLog Writes to Windows event log Write-EventLog -LogName Application -Source "MyScript" -EventId 3001 -Message "Script executed successfully"
Get-ADObject Searches AD for any object type Get-ADObject -Filter 'ObjectClass -eq "user" -or ObjectClass -eq "computer"' -SearchBase "DC=domain,DC=com"
Get-WmiObject Gets WMI object information Get-WmiObject -Class Win32_OperatingSystem | Select-Object Caption, Version, BuildNumber
Start-Transaction Starts a transaction block Start-Transaction; New-Item -Path "C:\test.txt" -UseTransaction; Complete-Transaction
Set-NetIPAddress Modifies IP address settings Set-NetIPAddress -InterfaceIndex 12 -IPAddress "192.168.1.100" -PrefixLength 24
Get-NetRoute Displays routing table Get-NetRoute | Where-Object DestinationPrefix -eq '0.0.0.0/0'
ConvertTo-SecureString Converts plain text to secure string $SecurePassword = ConvertTo-SecureString "Password123" -AsPlainText -Force
Get-AuthenticodeSignature Gets file signature information Get-AuthenticodeSignature -FilePath "C:\Windows\System32\calc.exe"
Get-WinEvent Gets events from event logs Get-WinEvent -LogName Security -MaxEvents 10 | Where-Object {$_.LevelDisplayName -eq 'Error'}
Get-ADReplicationPartnerMetadata Gets AD replication information Get-ADReplicationPartnerMetadata -Target "dc=contoso,dc=com" -Scope Server
Get-VM Lists virtual machines Get-VM | Where-Object {$_.State -eq 'Running'}
Get-PhysicalDisk Gets physical disk information Get-PhysicalDisk | Select-Object FriendlyName, MediaType, HealthStatus
Get-IISAppPool Lists IIS application pools Get-IISAppPool | Select-Object Name, State
New-WebApplication Creates a new web application New-WebApplication -Name "MyApp" -Site "Default Web Site" -PhysicalPath "C:\inetpub\wwwroot\MyApp" -ApplicationPool "MyAppPool"
Invoke-Sqlcmd Executes SQL queries Invoke-Sqlcmd -ServerInstance "ServerName" -Database "DBName" -Query "SELECT * FROM TableName"
Get-Mailbox Retrieves mailbox information Get-Mailbox -Identity "user@domain.com" | Select-Object DisplayName, Database
Add-DnsServerResourceRecordA Adds A record to DNS zone Add-DnsServerResourceRecordA -Name "host" -ZoneName "domain.com" -IPv4Address "192.168.1.100"
Get-RDSession Lists RDS sessions Get-RDSession -ConnectionBroker "broker.domain.com"
Start-WBBackup Starts Windows Server Backup Start-WBBackup -Policy (Get-WBPolicy)
Get-HotFix Lists installed Windows updates Get-HotFix | Sort-Object InstalledOn -Descending
Install-WindowsFeature AD-Domain-Services Installs AD Domain Services role Install-WindowsFeature -Name AD-Domain-Services -IncludeManagementTools
Install-ADDSForest Creates new AD forest Install-ADDSForest -DomainName "contoso.com" -InstallDNS -Force
Install-WindowsFeature DNS Installs DNS Server role Install-WindowsFeature -Name DNS -IncludeManagementTools
Install-WindowsFeature DHCP Installs DHCP Server role Install-WindowsFeature -Name DHCP -IncludeManagementTools
Install-WindowsFeature Web-Server Installs IIS Web Server role Install-WindowsFeature -Name Web-Server -IncludeManagementTools
Install-WindowsFeature Hyper-V Installs Hyper-V role Install-WindowsFeature -Name Hyper-V -IncludeManagementTools -Restart
Install-WindowsFeature RDS-Connection-Broker Installs RDS Connection Broker role Install-WindowsFeature -Name RDS-Connection-Broker -IncludeManagementTools
Install-WindowsFeature Failover-Clustering Installs Failover Clustering feature Install-WindowsFeature -Name Failover-Clustering -IncludeManagementTools
Get-WindowsFeature Lists all available Windows features Get-WindowsFeature | Where-Object InstallState -eq "Available"
Install-WindowsFeature File-Services Installs File Services role Install-WindowsFeature -Name FS-FileServer -IncludeManagementTools
Install-WindowsFeature Print-Services Installs Print Services role Install-WindowsFeature -Name Print-Server -IncludeManagementTools
Install-WindowsFeature NPAS Installs Network Policy Server Install-WindowsFeature -Name NPAS -IncludeManagementTools
Install-WindowsFeature WDS Installs Windows Deployment Services Install-WindowsFeature -Name WDS -IncludeManagementTools
Install-WindowsFeature WSUS Installs Windows Server Update Services Install-WindowsFeature -Name UpdateServices -IncludeManagementTools
Install-WindowsFeature ADCS-Cert-Authority Installs Certificate Authority role Install-WindowsFeature -Name ADCS-Cert-Authority -IncludeManagementTools
