Windows Server Basics

Understanding the fundamentals of Windows Server administration

Core Concepts

Domain Controller

The central authority for authentication and security in a Windows network. Think of it as the security guard of your network.

Active Directory

A database that stores information about network resources and users. Like a company's organizational chart and phone directory combined.

Group Policy

Rules and settings that control what users can do. Similar to setting rules and permissions for different departments.

Common Server Roles

File Server

Stores and manages access to files and folders.

Install-WindowsFeature -Name FS-FileServer

DHCP Server

Automatically assigns IP addresses to network devices.

Install-WindowsFeature -Name DHCP

DNS Server

Translates domain names to IP addresses.

Install-WindowsFeature -Name DNS

Print Server

Manages printers and print jobs across the network.

Install-WindowsFeature -Name Print-Server

Essential PowerShell Commands

Get-Service

Lists all services running on the server

Get-EventLog -LogName System -Newest 10

Shows the 10 most recent system events

Get-NetIPAddress

Displays all IP addresses configured on the server
