Understanding Network Basics

A simple guide to understanding how computer networks work

Basic Concepts

IP Address

Think of an IP address like a house address. Every device on a network has its own unique address (e.g., 192.168.1.1).

Subnet Mask

Like a postal code, it defines which part of the IP address represents the network and which part represents the host.

Gateway

The gateway is like the post office - it's where devices send data when communicating with other networks.

Network Devices

Router

Routes data between different networks, like a traffic controller directing cars between different roads.

Switch

Connects devices within the same network, like a power strip connecting multiple devices to power.

Firewall

Acts as a security guard, controlling what traffic is allowed in and out of your network.

Common Network Tools

ping

Tests if a device is reachable on the network.

ping 192.168.1.1

ipconfig/ifconfig

Shows your network configuration.

ipconfig /all
